# Rock Paper Scissors Game 🎮

An interactive and fun **Rock Paper Scissors game** where you can play against the computer and track your performance with a built-in score system.

## 🚀 Features

* 🎯 User vs Computer gameplay
* 🔢 Real-time score tracking (Wins, Losses, Draws)
* ⚡ Instant result display
* 🎲 Random computer choices
* 💡 Clean and responsive user interface

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎮 How to Play

1. Select **Rock**, **Paper**, or **Scissors**
2. The computer will randomly choose its move
3. The result will be displayed instantly:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. Scores will update automatically after each round

## 📊 Score System

* Tracks:

  * ✅ Wins
  * ❌ Losses
  * 🤝 Draws
* Helps users monitor their performance over multiple rounds
